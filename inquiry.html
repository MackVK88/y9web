Urban Population Overtime

By: Mack McGregor

Explanation

The web is home to near unlimited data that is right at your fingertip. The reason some webpages get more viewers than others, is not to do with the data, but rather the visual appeal. When looking at the Global Health Organization (GHO), the webpage features hundreds of different data tables, all stored within one page. The problem with this webpage, is that there is just rows and rows on the data table which could be sorted much differently. The data I will be using is one on urban overpopulation over time. Urban overpopulation is a problem as this creates an uneven population distribution which can lead to overcrowding, or can cause harm to the environment due to the Urban heat island effect. The next paragraph will explain the need for this data to have a basic HTML5 canvas.

Justification

HTML5 is a canvas used to draw graphics on a webpage. HTML5 is a good canvas to use for this data as it is supported by many different browsers. It also can create visually appealing graphs with medium code difficulty. To further prove, there is an id function that determines the width and height of the canvas. With all these as well as many different functions, HTML5 canvas is perfect to use with medium-low level code to create an awesome layout and webpage. It is important that this data becomes easier to read so people can understand the negatives surrounding urban overpopuation.

Target Audience

The target audience is towards people with good data such as the urban overpopulation, but has boring data, that is hard to read and digest. With basic HTML, CSS, And JavaScript code, and a proper canvas these web developers will be able to share their data with the world that is easy to understand and read. This code will help raise the issue of the data being shared.
